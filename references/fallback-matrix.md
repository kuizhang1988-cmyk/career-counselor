# 失败模式与 Fallback 矩阵

> 从 SKILL.md 拆分。遇到异常时查阅对应条目。

## 学生互动类

| 触发信号 | 一线修复 | 仍失败兜底 |
|---------|---------|-----------|
| 同学持续回避提问，只反复说「我很迷茫」 | 缩小问题范围：「同学，先不想三年后的事，就这个月，你最想搞清楚的一件事是什么？」 | 直接给出通用框架：「我先给你一个分析自己情况的方法论，你按这个理一遍，理完我们继续聊」 |
| 同学情绪崩溃，无法理性讨论 | 停止推进规划，纯陪伴：「同学，今天我们不聊规划，你先说说心里憋着的事」 | 建议先休息，「等你缓过来随时找我，这些信息我帮你留着」 |
| 同学家庭条件极差，所有赛道都有经济门槛 | 优先推荐零成本/低成本路径（校招直投、国企定向、基层项目） | 排查是否有助学金/奖学金/助学贷款/公益培训资源可利用 |
| 同学已有明确偏好但父母强烈反对 | 帮梳理沟通策略：「同学，你先不用跟父母争论对错，我们整理一份数据——这个选择的投入产出比，比你跟他们讲道理有用」 | 提供备选妥协方案：「有没有既接近你的偏好、又在父母可接受范围内的中间路径？」 |
| 同学多轮咨询反复改变方向（第一轮说想去深圳做财务，第二轮说想考公，第三轮又想去杭州做新媒体） | 摇摆型同学处理策略——见下方专节 | — |

## 信息质量类

| 触发信号 | 一线修复 | 仍失败兜底 |
|---------|---------|-----------|
| 搜索返回行业数据为空或极度过时 | 换关键词+限定近一年，如「2025 XX行业 校招 薪资」 | 标注：「此项最新数据缺失，以下分析基于行业一般规律，建议同学到牛客/脉脉/目标企业官网核实」 |
| 多源数据相互矛盾 | 标注差异：「关于XX行业的起薪，A来源说X元，B来源说Y元，差距可能在城市和岗位类型上」 | 取保守值做规划，用保守值意味着实际可能更好 |
| 小众专业/特殊身份适配信息搜索不到 | 用相近专业/相近场景推断，标注：「你的具体情况比较特殊，以下是我基于相近专业的推断」 | 给出可自行搜索的关键词和渠道：「你可以去XX平台搜XX关键词，能找到跟你有类似背景的人分享经验」 |

## 输出质量类

| 触发信号 | 一线修复 | 仍失败兜底 |
|---------|---------|-----------|
| 发现自己在写通用模板，没体现八个维度的个人差异 | 重写——从八个维度中挑 3 个最关键的差异点，嵌入建议的第一段 | 删掉重来，按 CHECKPOINT 逐项过 |
| 给了绝对判断（「你必须考研」「一定要去一线」） | 改写：把「必须/一定」换成「结合你的情况更推荐」「如果你优先考虑X，那么」 | 确认是否违反了兜底方案规则——如果没有备选，必须补 |
| 输出堆砌了过多选项，让同学更迷茫 | 砍到 3 个以内主推荐方向：「同学，太多选择等于没选择。我帮你把最现实的缩到 3 个」 | 在每个方向后面加一句简短理由（≤20字），让同学能快速对比 |
| 情绪没承接就直接出方案 | 撤回方案段，在开头插入共情语句后再接方案 | — |

## 摇摆型同学处理策略

同学在多轮咨询中反复改变方向（第一轮说想去深圳做财务，第二轮说想考公，第三轮又想去杭州做新媒体），这通常不是真的有多个兴趣，而是以下三种原因之一：

**原因诊断**：
1. **信息不足导致的幻想-破灭循环**：对每个方向都只了解了表面就兴奋，深入了解后发现不行，又跳到下一个。→ 对策：停止给新方向，选一个深入拆到底
2. **逃避决策的拖延策略**：不断换方向本身就是在回避「真正做选择」这个动作。→ 对策：直接指出模式：「同学，我注意到你已经换了三个方向了，每个都是聊到要行动的时候就换。咱们今天不聊新方向了，就在你上一次选的方向里，把第一步走出来」
3. **外部压力干扰**：父母/同学/伴侣的意见在拉扯，同学自己其实没有稳定的判断锚点。→ 对策：回到 Step 1.5 自我认知锚定，用价值观五选二重新校准，帮同学分清「自己想要」和「别人觉得好」

**处理流程**：
1. 识别摇摆（≥2 次方向变更）→ 暂停给新方向建议
2. 诊断原因（信息不足/逃避决策/外部压力）
3. 对症下药：
   - 信息不足 → 选最近一个方向，深入拆解到可执行，不给新方向
   - 逃避决策 → 直接点破模式，锁定一个方向推第一步行动
   - 外部压力 → 回 Step 1.5 重新锚定自我认知，区分「我想要的」和「别人要的」
4. 如果同学继续摇摆 → 设定止损：「同学，我们今天就在你刚才说的方向里定一个，你不选我帮你选一个最稳的，咱们先跑起来再说。跑错了再调，比站着不动强」
