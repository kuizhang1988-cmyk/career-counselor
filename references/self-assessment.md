# 自我认知触发工具

> 应届生说不清楚自己适合什么、喜欢什么、受不了什么是常态。在 Step 2 信息收集之前，先用这个工具帮同学找到自己的锚点。

## 使用时机

Step 0 判定为「纯框架」或「混合」后，在问八个维度之前，先扔这一组问题。同学如果已经能清晰描述自己，可以跳过。

## 三个锚点问题

**问题 1：做什么事的时候，你最容易忘记时间？**

这个问题的本质是找「心流」。不是问「你喜欢什么」，因为喜欢看电影不等于适合剪片子。心流是——做过的事里，哪一类让你完全沉浸、水都忘了喝。不一定跟专业相关。有人是在社团做海报时，有人是在帮同学选课做攻略时，有人是在给朋友分析人生选择时。

提取技巧：同学说「我喜欢打游戏」→ 追问「是喜欢跟人对战还是一个人闯关？如果是跟人，你在团队里通常扮演什么角色？」——从娱乐行为里能挖出社交偏好和决策模式。

**问题 2：什么样的工作环境会把你逼疯？**

消极偏好比积极偏好更容易准确识别。让人说出「我喜欢什么」很难，让ta说出「我绝对不能接受什么」几乎人人都能秒答。

常见答案及对应的排除清单：
- 「受不了天天加班」 → 排除投资银行/四大审计/互联网大小周企业
- 「受不了跟陌生人说话」 → 排除销售/客户经理/猎头/商务BD
- 「受不了重复劳动」 → 排除流水线式审核/初级财务对账/客服
- 「受不了办公室政治」 → 优先技术岗/中小创业公司/自由职业
- 「受不了规则太多」 → 排除体制内/国企/合规岗

**问题 3：别人总觉得你最强的能力是什么？**

不要问「你有什么能力」——刚毕业的学生对自己能力的认知严重失真。问「别人觉得」。因为别人夸你的话你记不住，但重复得最多的大概率是真的。

常见答案解码：
- 「别人说我靠谱」 → 执行力强、交付可靠 → 适配项目管理/运营/体制内
- 「别人说我反应快」 → 学习力强、善于临场 → 适配互联网/咨询/销售
- 「别人说我人缘好」 → 社交力强、人脉经营 → 适配商务/HR/市场
- 「别人说我想得细」 → 分析力强、风险敏感 → 适配风控/审计/法务/数据分析

## 价值观五选二

问完三个锚点后，扔这个：

> 同学，最后问你一个关键问题。下面五个东西，你只能选两个——不是「都重要」，是真的只能选两个，你选哪两个？
> 1. 钱（前五年收入尽量高）
> 2. 稳（不太可能失业或被裁）
> 3. 长（能力成长速度快，越老越值钱）
> 4. 闲（不加班，有自己的生活）
> 5. 光（社会地位高，父母有面子）

**五选二的决策意义**：

| 选择组合 | 推荐方向 | 示范 |
|---------|---------|------|
| 钱 + 长 | 高成长行业核心岗 | 互联网研发/量化金融/咨询 |
| 钱 + 闲 | 高时薪稀缺技能 | 远程技术岗/自由职业/投资 |
| 稳 + 闲 | 体制内/国企 | 公务员/事业编/地方国企 |
| 稳 + 光 | 公务员/头部央企 | 选调生/国家部委/央企总部 |
| 长 + 光 | 专业服务/学术 | 律师/医生/高校/研究院 |
| 稳 + 长 | 体制内技术岗 | 体制内IT/工程师/科研院所 |
| 钱 + 光 | 头部企业+公众岗位 | 大厂产品/头部投行/明星创业 |
| 闲 + 光 | 矛盾组合，需追问 | 通常不存在，引导重新选 |

## 输出格式

自我认知梳理完成后，在进入 Step 2 之前，用一句话反馈给同学：

> 好，我帮您提炼一下——您是一个［在XX场景下容易沉浸］、［最受不了XX］、［别人觉得您XX能力特别强］的人。在钱稳长闲光里，您选了［X］和［Y］。我们带着这个自我认知，来看您的具体情况。

这句话的作用是锚定——让同学感觉「你确实听懂了我是谁」，再去问院校专业家庭条件，ta的配合度完全不一样。
